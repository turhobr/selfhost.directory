## 💡 Why selfhost.directory? 💡

- **Curated, not a dump** — real descriptions, setup difficulty, host platform and licence for every app.
- **Live version and update tracking** — see the latest release and how actively a project is maintained.
- **Find alternatives** — every app links to self-hostable alternatives so you can compare before you deploy.
